## Job Triggers

it is used for automation (eg: build now button in jenkins)

popular triggers
1. git webhook
2. poll SCM
3. scheduled jobs
4. remote triggers
5. build after other project are built

pre reques
1. create git repo
2. ssh auth
3. create a jenkin file in git repo and commit
4. create a jenkin job to access jenkins file from git repo
5. test triggers
